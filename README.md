# chepelovskyi.com

Персональный сайт Андрея Чепеловского — минималистичная статическая платформа
с собственной админкой поверх Firebase.

## Структура

- `index.html` — главная (герой-цитата, последние записки, стихи, YouTube)
- `path.html` — путь автора (длинный текст)
- `notes.html` + `notes/` — записки (короткие тексты)
- `poetry.html` + `poetry/` — стихи
- `note.html`, `poem.html` — универсальные страницы одной записки/стиха из Firestore
- `youtube.html` — канал «Стихи вслух»
- `contact.html` — форма обратной связи (Web3Forms)
- `admin.html` — админка (скрытая, только для владельца)
- `css/main.css`, `css/admin.css` — стили
- `js/main.js` — навигация, поиск, форма, скрытая точка входа в админку
- `js/content-loader.js` — подставляет контент из Firestore в публичные страницы
- `js/admin.js` — логика админки
- `firebase-config.js` — инициализация Firebase (Auth + Firestore)
- `firestore.rules` — правила доступа Firestore
- `sw.js` — Service Worker для офлайн-кеша

## Как редактировать сайт

Весь контент хранится в Firestore. Редактируется через админку.

### Открыть админку

1. Адрес напрямую: `https://chepelovskyi.com/admin.html`
2. Или тройной клик по году в футере любой страницы сайта
3. Или горячая клавиша `Ctrl+Shift+A` (на Mac: `Cmd+Shift+A`)

### Первая настройка

1. Зайти в [Firebase Console](https://console.firebase.google.com/)
   → проект `chepelovskyi-site` → **Authentication**.
2. Включить способ входа **Email/Password**.
3. Создать одного пользователя (свой email + пароль) через
   **Users → Add user**.
4. Скопировать его **UID**.
5. Открыть `firestore.rules`, заменить `OWNER_UID` на свой UID.
6. Опубликовать правила: **Firestore → Rules → Publish** (или
   `firebase deploy --only firestore:rules`).
7. Готово. Заходи на `/admin.html` и входи своими email+паролем.

### Разделы админки

- **Главная** — герой-цитата, подпись, кнопка.
- **Путь** — заголовок, эпиграф, длинный текст (поддерживает `##` и `>`).
- **Записки** — список, создание, редактирование, удаление.
- **Стихи** — список, создание, редактирование, удаление.
- **YouTube** — заголовок страницы, список видео (по ID).
- **Контакт** — заголовок, текст над и под формой.
- **История** — последние 30 снимков, откат одним кликом.

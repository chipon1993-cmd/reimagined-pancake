# chepelovskyi.com

Персональный сайт Андрея Чепеловского — минималистичная статическая платформа.

## Структура

- `index.html` — главная (герой-цитата, последние записки, стихи, YouTube)
- `path.html` — путь автора (длинный текст)
- `notes.html` + `notes/` — записки (короткие тексты)
- `poetry.html` + `poetry/` — стихи
- `youtube.html` — канал «Стихи вслух»
- `contact.html` — форма обратной связи (Web3Forms)
- `css/main.css` — единая таблица стилей
- `js/main.js` — навигация, поиск, форма
- `sw.js` — Service Worker для офлайн-кеша
- `firebase-config.js` — Firebase (сохранён как внешний сервис; пока не используется фронтендом)

## Контент

Весь текст сейчас — заполнитель. Автор заменяет его своими словами прямо в HTML-файлах.

## Стек

Статический HTML/CSS/JS без фреймворков. Хостинг: GitHub Pages. Форма: Web3Forms. Домен: `chepelovskyi.com`.
